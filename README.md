# Employe management system
 
A ** GUI-based Employe Management System ** developed in **python** using **Tkinter** for the interface and **MySQL (pymysql)** as the database backend. this system allows users to **add, update, delete, view employe records, and check employe salary** in an intuitive graphical interface




## features 

**Add employee** : Insert employe records with ID, name, mobile number, department, and salary.
**view employee** : Display all employe records in a table format using 'Treeview' .
